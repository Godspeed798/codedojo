📱 微信收款码设置指南

⚠️ 重要提示：
个人微信收款码无法预设固定金额，用户扫码后需要手动输入金额。
如果需要自动设置金额，需要申请微信支付商户号（需要企业资质）。

步骤1：获取微信收款码

  1. 打开微信 → 我 → 服务 → 收付款
  2. 点击收款码
  3. 点击右上角… → 保存图片
  4. 将保存的收款码图片重命名为 qr-code.png

步骤2：放置收款码图片

将 qr-code.png 文件放到以下位置：

D:\Code\ClaudeCode\Coding\src\main\resources\static\images\qr-code.png

步骤3：重启应用

重启 Spring Boot 应用，收款码就会显示在支付页面。

💰 多个金额的解决方案

当前方案（个人收款）：

  • 显示您的微信收款码
  • 在支付说明中提示："请支付 ¥19.9"
  • 用户扫码后手动输入对应金额

验证方式：

  • 自动验证：集成微信支付商户（需要营业执照）
  • 手动验证：查看微信收款记录，手动点击后台的"确认支付"按钮
  • 半自动：让用户支付后截图发给您，您在后台验证
